# Idea Generator Agent

Generate creative writing ideas from analyzed content.

## Task

1. Read ContentSummary from `contentItem/{id}/summary.json` (generated by content-analyzer)
2. Read user-profile.json for personalization
3. Generate hooks (5 types: emotional, curiosity, controversy, statistic, story)
4. Suggest narrative angles with relevance scores
5. Formulate exploratory questions (4 types: analytical, practical, philosophical, comparative)
6. Write output to: `contentItem/{id}/ideas.json`

## Output

Write to: `contentItem/{id}/ideas.json`

Return WritingIdeas JSON with:
- hooks: Array of writing hooks with type and text
- angles: Array of narrative angles with relevance scores
- questions: Array of exploratory questions by type

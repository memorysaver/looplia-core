# Content Analysis Skill

This skill helps analyze content and extract:
- Key themes and topics
- Writing hooks and angles
- Exploratory questions
- Article outlines

## Usage

Invoke this skill when the user wants to transform raw content into writing materials.

## Workflow

1. **Summarization**: Extract key insights, create headline and TL;DR
2. **Ideation**: Generate hooks, angles, and questions
3. **Outlining**: Create structured article outline

## Output Formats

All outputs follow the Looplia core schemas:
- ContentSummary for summaries
- WritingIdeas for creative expansion
- OutlineSection[] for article structure
